# Aliasist Platform

![Aliasist banner](https://raw.githubusercontent.com/aliasist/aliasistabductor/master/images/aliasist_banner_orbit.png)

Aliasist Platform is the private suite repo for the main product surface.

It combines:

- a single portal for navigation
- domain apps for data, earth, and space
- one edge API layer
- shared UI and retrieval packages

The repo keeps the suite aligned as one system without exposing the implementation details here.
